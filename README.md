# Mupen64PlusU
A Mupen64Plus-Next port for Wii U in GX2 video-backend
Finally, after years of waiting, you could now natively emulate N64 games on your beloved console!
Supports EVERY SINGLE N64 GAME/MOD (it's a fork of Mupen64Plus-Next!)

FEATURES:

60FPS patch, full Wii U GamePad support, no need to configure buttons!
Save states!
ROMs in wiiu/apps/Mupen64PlusU/roms

Enjoy!
